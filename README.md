# AWS Cloud Monitoring & Cost Governance

## 📌 Overview
This project demonstrates how to monitor AWS resources and optimize cloud costs using AWS CloudWatch, SNS, and Python (Boto3).

## ⚙️ Services Used
- AWS EC2
- AWS S3
- AWS CloudWatch
- AWS SNS
- AWS Budgets
- AWS Cost Explorer
- Python (Boto3)

## 🚀 Features
- Real-time monitoring dashboards using CloudWatch
- Automated alerts using SNS notifications
- Cost tracking and budget alerts
- Python automation for cost analysis

## 📊 Architecture
EC2 / S3 → CloudWatch → SNS Alerts → Email Notification  
                         ↓  
                    Python (Boto3) → Cost Monitoring
